# Avatar Packs

Avatar packs are curated asset bundles grouped by style family.

Phase 1 packs should be 2D and parts-based.

Expected future structure:

```text
avatar-packs/
  anime_soft/
    manifest.json
    parts/
      face_base/
      hair_front/
      hair_back/
      brows/
      eyes/
      nose/
      mouth_set/
      accessories/
  action_figure/
    manifest.json
    parts/
      ...
```

Each pack should be consumable by the avatar descriptor schema.
