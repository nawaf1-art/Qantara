# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

The code applies in all project spaces — GitHub issues, pull requests, discussions, and any community channels linked from the Qantara README — and when individuals are representing the project publicly.

## Enforcement

If you experience or witness unacceptable behavior, please report it through GitHub's private vulnerability reporting flow (see `SECURITY.md`) or by contacting the maintainer listed on the repository owner's GitHub profile. Reports are handled confidentially.

Community leaders are responsible for clarifying and enforcing standards, and will take fair corrective action in response to any behavior they deem inappropriate.
